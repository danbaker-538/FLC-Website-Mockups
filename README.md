# Fun Loving Company - Website Design Mockups

Three distinct design directions for the FLC website redesign. Open each HTML file in a browser to preview.

---

## Mockup 1: Bold & Editorial
**File:** `mockup-1-bold-editorial.html`

**Style:** High-impact, gradient-heavy, Psilly Goose-inspired
- Large gradient hero (peach/orange)
- Floating, animated product shot
- Scrolling benefit ticker
- Bold editorial headlines
- Strong visual hierarchy

**Best for:** Making a big splash, younger audience appeal, standing out in the market

---

## Mockup 2: Clean & Premium
**File:** `mockup-2-clean-premium.html`

**Style:** Sophisticated, minimal, trustworthy
- Clean white space
- Subtle product showcase with soft gradient accent
- Trust metrics prominently displayed
- Step-by-step "How It Works" section
- Professional, refined typography

**Best for:** Building credibility with 30+ audience, premium positioning, health-conscious consumers

---

## Mockup 3: Playful & Social
**File:** `mockup-3-playful-social.html`

**Style:** Fun, community-focused, energetic
- Announcement bar for urgency
- Playful animations (wiggling bottle, floating badges)
- Navy + orange contrast sections
- Strong community/newsletter focus
- Social proof and perks emphasis

**Best for:** Building community, email list growth, emphasizing the social/fun positioning

---

## Design Elements Used

**Colors (kept consistent per brand):**
- Primary Orange: #E8724A
- Cream/Off-white backgrounds
- Navy accents for contrast

**Fonts (Google Fonts alternatives to approximate existing brand):**
- Mockup 1: Playfair Display + Inter
- Mockup 2: DM Serif Display + DM Sans
- Mockup 3: Fraunces + Plus Jakarta Sans

**Sections Included:**
- Homepage hero with product
- Benefits/trust indicators
- Product showcase
- Learn section (Kanna education)
- News grid
- Newsletter/community signup
- Footer

---

## How to View

1. Open any `.html` file directly in your browser (Chrome, Safari, Firefox)
2. Each mockup has a label in the bottom-right corner identifying which design it is
3. The bottle image is included in the `assets` folder
